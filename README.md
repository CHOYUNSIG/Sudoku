# sudoku
스도쿠 게임 앱 (Windows)

편안한 주말 오후의 느낌으로 스도쿠를 즐기세요!

## Features

1. 편리하고 미적인 인터페이스
2. 문제 자동/수동 생성 및 공유
3. 진행 상황 저장
4. 기록 세우기
5. 힌트
